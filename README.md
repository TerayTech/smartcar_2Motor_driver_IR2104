通用两轮电机驱动板
============

本驱动板为通用四轮组驱动板，使用IR2104驱动芯片构成双H桥，驱动双电机。  
在最后测试中无大问题，但是这种结构的驱动容易烧毁12V升压芯片，如果给直立车使用需要特别注意。
